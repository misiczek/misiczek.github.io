JavaScript - wprowadzenie

JavaScript to język programowania skryptowego najczęściej stosowany na stronach internetowych do zapewniania ich interaktywności, obsługi zdarzeń, walidacji danych wprowadzanych w formularzach oraz tworzenia złożonych efektów wizualnych. Działa on po stronie klienta, jest wykonywany lokalnie na komputerze użytkownika strony internetowej. Jest językiem interpretowanym, czyli takim, który nie wymaga kompilacji (zamiany kody źródłowego na binarny język maszynowy). Kod JavaScriptu jest wczytywany, interpretowany i wykonywany przez interpreter języka zawarty w przeglądarce internetowej.

Zmienne - let, var

Zmienną możemy zadeklarować za pomocą słowa kluczowego let (istnieje również starsza metoda - var, jednak obecnie nie zaleca się jej używania).

Przykład:

Deklaracja zmiennej bez przypisania jej wartości domyślnej: 
let zmienna1;
Deklaracja zmiennej z przypisaniem jej wartości domyślnej: 
let zmienna1 = 1;
let zmienna2 = "tekst";
Jeśli po zadeklarowaniu zmiennej chcemy przypisać jej inną wartość, nie używamy ponownie słowa let, ponieważ służy ono jedynie do jej deklaracji. 
let zmienna = 1;
zmienna = 2;

Typy danych

Liczba (number), - w JavaScipcie jest tylko jeden typ zmiennej liczbowej, zarówno dla liczb całkowitych, jak i zmiennoprzecinkowych: 
let zmienna = 1;
Ciąg znaków (string): 
let zmienna = "tekst";
Wartość logiczna (boolean), prawda: 
let zmienna = true;
Wartość logiczna (boolean), fałsz: 
let zmienna = false;
Oznaczenie braku wartości: 
let zmienna = null;
Tablica (array), (omówione dalej): 
let zmienna = [element1, ..., elementX];

Stałe - const

Wartość stałej, w przeciwieństwie do zmiennej, nie może zostać zmieniona w trakcie wykonywania kodu.

Przykład:

Deklaracja stałej z przypisaniem jej liczby 1: 
const stala = 1;
Deklaracja stałej z przypisaniem jej ciągu znaków "tekst": 
const stala = "tekst";

Wypisanie danych do konsoli - console.log

console.log wypisuje dane do konsoli przeglądarki internetowej.

Konsola dostępna jest w dowolnej przeglądarce po kliknięciu prawego przycisku myszy na stronie internetowej i wybraniu opcji "Zbadaj"/"Wykonaj inspekcję". Następnie należy odnaleźć zakładkę "Console"/"Konsola".

Przykład:

Wypisanie tekstu "Hello world!": 
console.log("Hello world!");
Wypisanie zawartości zmiennej: 
let zmienna = "Tekst";
console.log(zmienna);

Okna dialogowe przeglądarki - alert, prompt, confirm

alert - Informacja dla użytkownika, posiada jedynie przycisk "OK".
prompt - Okno, które służy do wprowadzania danych.
confirm - Okno wyposażone w dwa przyciski: "OK" oraz "Anuluj", prosi użytkownika o potwierdzenie.

Przykład:

alert: 
alert("Oto komunikat.");
Skrypt, który prosi użytkownika o podanie swojego imienia (prompt), a następnie zwraca je używając alert. 
let imie = prompt("Podaj imię:");
alert(imie);
Skrypt, który prosi użytkownika o potwierdzenie swojego wieku (confirm), a następnie zwraca jego wybór używając console.log. Jeśli użytkownik wybierze "OK", do konsoli zostanie zwrócona wartość "true", natomiast jeśli wybierze opcję "Anuluj", zwrócona zostanie wartość "false". 
let wiek = confirm("Czy masz 18 lat?");
console.log(wiek);

Instrukcja warunkowa - if

If sprawdza, czy dany warunek jest prawdziwy, jeżeli tak to wykonuje zawarte polecenie.

else if - Jeśli poprzedni warunek nie został spełniony, else if sprawdza nowy warunek, jeżeli jest prawdziwy, wykonuje go, w przeciwnym wypadku przechodzi do następnego else if bądź else.

else - Wykonuje kod, jeśli wszystkie poprzednie warunki nie zostały spełnione.

Składnia:

Tylko if: 
if(warunek){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek
}
if, else if, else: 
if(warunek1){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek1
}
else if(warunek2){
    //kod, który wykonuje się, jeśli warunek1 nie zostanie spełniony i warunek2 będzie prawdziwy
}
else{
    //kod, który wykonuje się, jeśli wszystkie warunki nie zostały spełnione
}

Przykład:

Kod, który sprawdza, czy liczba jest dodatnia, ujemna lub równa 0: 
let liczba = 1;

if(liczba > 0){
    alert("Liczba jest dodatnia.");
}
else if(liczba < 0){
    alert("Liczba jest ujemna.");
}
else{
    alert("Liczba jest równa 0.");
}

Operatory logiczne - &&, ||

Operatory logiczne pozwalają łączyć wiele warunków w jednej instrukcji warunkowej.

&& - i/oraz, w przypadku, gdy mamy dwa warunki, oba muszą zostać spełnione.

|| - lub, w przypadku, gdy mamy dwa warunki, przynjamniej jeden musi zostać spełniony.

Składnia:

&&: 
if(warunek1 && warunek2){
    //kod, który wykona się po spełnieniu dwóch warunków.
}

||: 
if(warunek1 || warunek2){
    //kod, który wykona się po spełnieniu przynajmniej jednego warunku.
}

Przykład:

&&: Skrypt, który sprawdza, czy wartość zmiennej jest większa od 1 i mniejsza od 10: 
let liczba = 5;

if(liczba > 1 && liczba < 10){
    alert("Prawidłowa liczba");
}
else{
    alert("Nieprawidłowa liczba");
}
||: Skrypt, który sprawdza, czy liczba jest mniejsza od 1 lub większa od 10: 
let liczba = 5;

if(liczba < 1 || liczba > 10){
    alert("Prawidłowa liczba");
}
else{
    alert("Nieprawidłowa liczba");
}

Pętle - while, do while, for

Pętle umożliwiają wielokrotne wykonanie tego samego fragmentu kodu.

while - Wykonuje kod, jeśli dany warunek jest spełniony.

do while - Wykonuje kod przynajmniej raz. Jeśli warunek jest spełniony, pętla wykonuje się ponownie.

for - Pętla, którą najczęściej wykorzystuje się, gdy chcemy wykonać kod określoną ilość razy.

Składnia:

while: 
while(warunek){
    //Kod wykonywany, dopóki warunek jest prawdziwy.
}
do while: 
do{
    //Kod, który wykonuje się przynajmniej raz.
    //Jeśli warunek jest prawdziwy, wykonuje się dalej.
}while(warunek);
for:

inicjalizacja_zmiennej - Deklaracja oraz nadanie wartości początkowej zmiennej (najczęściej 0). Zmienna ta pełni rolę licznika pętli.

warunek - Pętla będzie się wykonywać, dopóki ten warunek jest spełniony. Zazwyczaj jest to po prostu warunek, że licznik musi być mniejszy od podanej przez nas liczby, która określa, ile razy pętla ma się wykonać.

polecenie - Tutaj określamy, co będzie się działo z licznikiem przy każdym okrążeniu pętli. Najczęściej jest to inkrementacja, czyli zwiększenie wartości zmiennej o 1.

for(inicjalizacja_zmiennej; warunek; polecenie){
    //Kod do wykonania.
}

Przykład:

while:

Pętla przestanie się wykonywać, gdy użytkownik poda poprawną odpowiedź na działanie 2+2:

let odp;
while(odp != 4){
    odp = prompt("Ile to jest 2+2?");
}

do while:

Ten sam skrypt co wcześniej, ale z wykorzystaniem pętli do while:

let odp;
do{
    odp = prompt("Ile to jest 2+2?");
}while(odp != 4);

for:

Pętla, która wykona się 5 razy, za każdym razem wypisując do konsoli aktualną wartość zmiennej i. W konsoli zobaczymy liczby od 0 do 4 (a nie 5), ponieważ pętla rozpoczyna od wartości 0, a ostatnią możliwą wartością jest 4 (warunek i < 5).

for(let i = 0; i < 5; i++){
    console.log(i);
}

Tablice

Tablice są zbiorami danych. Mogą przechowywać dane wszelkiego, dostępnego w JS typu (nawet inne tablice). Deklarujemy je tak samo jak zmienne i stałe.

Składnia:

Deklaracja tablicy: 
const tablica1 = [element1, ..., elementX];

Przykład:

let lista_zakupow = ["Mleko", "Chleb", "Jajka", "Sok", "Masło"];

Tablicę stałą możemy modyfikować (omówione dalej) tak samo jak zmienną, lecz nie możemy do niej przypisać innej tablicy tzn.:

const tablica1 = [1,2,3];
let tablica2 = [4,5,6];
//Linijka poniżej nie wykona się i wypisze w konsoli błąd.
tablica1 = tablica2;

Metody pracy z tablicami i modyfikacja tablic

Podstawowe metody:

Zwrócenie długości tablicy (liczby jej elementów): 
let dlugosc = tablica.length;
Zwrócenie tablicy przekształconej w ciąg znaków: 
let tablicaString = tablica.toString();
Wstawianie elementu na końcu tablicy: 
tablica.push(element);
Wstawianie elementu na początku tablicy: 
tablica.unshift(element);
Usuwanie ostatniego elementu tablicy: 
tablica.pop();
Usuwanie pierwszego elementu tablicy: 
tablica.shift();
Łączenie tablic: 
let polaczoneTablice = tablica1.concat(tablica2);
Sortowanie elementów tablicy: 
tablica.sort();
Odwrócenie kolejności elementów w tablicy: 
tablica.reverse();

Metoda splice:

Metoda splice służy do usuwania, dodawania lub zamieniania elementów w tablicy.

Składnia:

indeks - Indeks elementu, od którego chcemy zacząć edytować tablicę.

liczba - Liczba elementów, którą chcemy usunąć od elementu z podanym przez nas wcześniej indeksem początkowym (włącznie z nim!).

element1, ..., elementX - Elementy, które chcemy dodać.

tablica.splice(indeks, liczba, element1, ..., elementX);

Przykład:

Tablica: 
let lista_zakupow = ["Mleko", "Chleb", "Jajka", "Sok", "Masło"];
Usuwanie wszystkich elementów z tablicy:

Został tutaj zadeklarowany jedynie indeks elementu, od którego chcemy zacząć, w tym przypadku od samego początku tablicy. Jeśli nie wprowadziliśmy kolejnego argumentu, czyli liczby elementów, które chcemy usunąć, metoda splice domyślnie usuwa wszystkie następne elementy.

lista_zakupow.splice(0);
Usuwanie wszystkiego, zaczynając od drugiego elementu (indeks 1): 
lista_zakupow.splice(1);
//W tym przypadku w tablicy zostanie jedynie element "Mleko".
Usuwanie jednego elementu, zaczynając od elementu trzeciego: 
lista_zakupow.splice(2, 1);
//W tym przypadku tablica będzie miała postać ["Mleko", "Chleb", "Sok", "Masło"]
Wstawianie na początku tablicy elementów "Piwo", "Gazeta":

Nowe elementy zostały wstawione w miejsce elementu pierwszego, lecz nie został on usunięty ze względu na drugi argument o wartości 0.

lista_zakupow.splice(0, 0, "Piwo", "Gazeta");
//Tablica: ["Piwo", "Gazeta", "Mleko", "Chleb", "Jajka", "Sok", "Masło"]
Wstawianie elementów "Piwo", "Gazeta", zaczynając od elementu trzeciego: 
lista_zakupow.splice(2, 0, "Piwo", "Gazeta");
//Tablica: ["Mleko", "Chleb", "Piwo", "Gazeta", "Jajka", "Sok", "Masło"]
Usunięcie dwóch elementów i wstawienie "Piwo", "Gazeta" zaczynając od elementu trzeciego: 
lista_zakupow.splice(2, 2, "Piwo", "Gazeta");
//Tablica: ["Mleko", "Chleb", "Piwo", "Gazeta", "Masło"]

Metoda concat - łączenie tablic

Metoda concat pozwala na łączenie tablic oraz dodawanie do nich nowych elementów.

Składnia:

Złączenie tablic:

zlaczoneTablice - Tablica, do której przypiszemy wartość złączonych tablic - tablica1 i tablica2.

const zlaczoneTablice = tablica1.concat(tablica2);

Złączenie tablic i dodanie elementów:

tablicaPlusElementy - Tablica, do której przypiszemy wartość złączonej tablicy1 z elementami.

const tablicaPlusElementy = tablica1.concat(element1, ..., elementX);

Przykład:

Łączenie tablic: 
const lista_zakupow1 = ["Mleko", "Chleb"];
const lista_zakupow2 = ["Jajka", "Sok"];

const zlaczoneListy = lista_zakupow1.concat(lista_zakupow2);

//Tablica zlaczoneListy będzie miała postać: ["Mleko", "Chleb", "Jajka", "Sok"]
Łączenie tablicy i elementów: 
const lista_zakupow = ["Mleko", "Chleb"];  
const lista_zakupow2 = lista_zakupow.concat("Jajka", "Sok");
        
//Tablica lista_zakupow2 będzie miała postać: ["Mleko", "Chleb", "Jajka", "Sok"]

Tablica: 
const imiona = ["Tomasz", "Karol", "Natalia"];

Funkcje

Funkcje są blokami kodu, który może zostać wykorzystany wielokrotnie w całym skrypcie. Zwarty w funkcji kod jest wykonywany tylko po jej wywołaniu.

W JavaScipcie występują dwa typy funkcji - klasyczne i strzałkowe.

Funkcje klasyczne

Ten rodzaj funkcji w budowie przypomina te z innych języków programowania.

Składnia:

Deklaracja funkcji: 
function nazwaFunkcji(parametry){
    //Kod, który funkcja ma wykonać.
}
Wywołanie funkcji: 
nazwaFunkcji(parametry);

Przykład:

Funkcja, która przyjmuje imię oraz nazwisko jako parametry i wypisuje do konsoli przywitanie: 
function powiedzCzesc(imie, nazwisko){
    console.log("Cześć", imie, nazwisko + "!");
}

powiedzCzesc("Gniewomir", "Bąk");
return - Określa wartość, którą funkcja ma zwrócić. Jeśli wywołanie funkcji przypiszemy do zmiennej, to wtedy wartość zwrócona zostanie wpisana do tej zmiennej. Instrukcja return nie jest obowiązkowa, lecz zazwyczaj chcemy, aby funkcja zwracała jakąś wartość.

Funkcja, która przyjmuje parametry w postaci dwóch liczb, a następnie zwraca wynik ich dodawania:

function dodaj(a, b){
    return a+b;
}

let wynik = dodaj(2,4);

console.log(wynik);
//Do konsoli zostanie wypisana zawartość zmiennej wynik, czyli 6.

Funkcje strzałkowe

Funkcje strzałkowe są skróconym zapisem funkcji klasycznej.

Składnia:

Deklaracja funkcji bez parametrów: 
let nazwaFunkcji = () => //Kod do wykonania;
Deklaracja funkcji z jednym parametrem: 
let nazwaFunkcji = parametr => //Kod do wykonania;
Deklaracja funkcji z wieloma parametrami: 
let nazwaFunkcji = (parametr1, ..., parametrX) => //Kod do wykonania;
Funkcja strzałkowa z wieloma wierszami kodu: 
let nazwaFunkcji = () =>{
    //Pierwszy wiersz kodu.
    //Dowolna liczba wierszy kodu.
}

Przykład:

Funkcja, która przyjmuje imię oraz nazwisko jako parametry i wypisuje do konsoli przywitanie: 
let powiedzCzesc = (imie, nazwisko) => console.log("Cześć", imie, nazwisko + "!");
powiedzCzesc("Gniewomir", "Bąk");

Uwaga: W funkcjach strzałkowych nie musimy (choć możemy) stosować instrukcji return, aby zwrócić wartość.

Funkcja, która przyjmuje parametry w postaci dwóch liczb, a następnie zwraca wynik ich dodawania:

let dodawanie = (a,b) => a+b;
let wynik = dodawanie(2,4);
console.log(wynik);
//Do konsoli zostanie wypisana zawartość zmiennej wynik, czyli 6.
