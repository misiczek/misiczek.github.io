JavaScript - wprowadzenie

JavaScript to język programowania skryptowego najczęściej stosowany na stronach internetowych do zapewniania ich interaktywności, obsługi zdarzeń, walidacji danych wprowadzanych w formularzach oraz tworzenia złożonych efektów wizualnych. Działa on po stronie klienta, jest wykonywany lokalnie na komputerze użytkownika strony internetowej. Jest językiem interpretowanym, czyli takim, który nie wymaga kompilacji (zamiany kody źródłowego na binarny język maszynowy). Kod JavaScriptu jest wczytywany, interpretowany i wykonywany przez interpreter języka zawarty w przeglądarce internetowej.

Zmienne - let, var

Zmienne możemy deklarować za pomocą słów kluczowych let i var. Zaleca się jednak korzystanie z let, gdyż jest to nowszy standard deklarowania zmiennej w JS.

Przykład:

Zadeklarowanie zmiennej bez przypisania jej wartości domyślnej: 
let zmienna1;
var zmienna2;
Zadeklarowanie zmiennej z przypisaniem jej wartości domyślnej: 
let zmienna1 = 1;
var zmienna2 = "tekst";

Typy danych

Liczba (number), (w JavaScipcie jest tylko jeden typ zmiennej liczbowej, zarówno dla liczb całkowitych jak i zmiennoprzecinkowych): 
let zmienna1 = 1;
Ciąg znaków (string): 
let zmienna2 = "tekst";
Wartość logiczna (boolean), prawda: 
let zmienna3 = true;
Wartość logiczna (boolean), fałsz: 
let zmienna4 = false;
Oznaczenie braku wartości: 
let zmienna5 = null;
Tablica (array), (omówione dalej): 
let zmienna6 = [element1, ..., elementX];

Stałe - const

Przykład: 

const stala1 = 1;
const stala2 = "tekst";

Wypisanie danych do konsoli - console.log

Przykład:

Wypisanie tekstu "Hello world!": 
console.log("Hello world!");
Wypisanie zawartości zmiennej: 
let zmienna = "Tekst";
console.log(zmienna);

Okna dialogowe przeglądarki - alert, prompt, confirm

alert - informacja dla użytkownika, posiada jedynie przycisk "OK".
prompt - okno, które może posłużyć do wprowadzania danych.
confirm - okno, wyposażone w dwa przyciski: "OK" oraz "Anuluj", prosi użytkownika o potwierdzenie.

Przykład:

alert: 
alert("Oto komunikat.");
Skrypt, który prosi użytkownika o podanie swojego imienia (prompt) a następnie zwraca je używając alert: 
let imie = prompt("Podaj imię:");
alert(imie);
Skrypt, który prosi użytkownika o potwierdzenie swojego wieku (confirm) a następnie zwraca jego wybór używając console.log: 
let wiek = confirm("Czy masz 18 lat?");
console.log(wiek);
Jeśli użytkownik wybierze "OK" do konsoli zostanie zwrócona wartość "true", natomiast jeśli wybierze opcję "Anuluj" zostanie zwrócona wartość "false"

Instrukcja warunkowa - if

if - sprawdza, czy dany warunek jest prawdziwy, jeżeli tak to wykonuje zawarte polecenie.

else if - jeśli poprzedni warunek nie został spełniony, else if sprawdza nowy warunek, jeżeli jest prawdziwy, wykonuje go, jeżeli jest fałszywy, przechodzi do następnego else if bądź else.

else - wykonuje kod, jeśli wszystkie poprzednie warunki nie zostały spełnione.

Składnia:

Tylko if: 
if(warunek){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek
}
if, else if, else: 
if(warunek1){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek1
}
else if(warunek2){
    //kod, który wykonuje się, jeśli warunek1 nie zostanie spełniony i warunek2 zostanie spełniony
}
else{
    //kod, który wykonuje się, jeśli wszystkie warunki nie zostaną spełnione
}

Przykład:

Kod, który sprawdza, czy liczba jest dodatnia, ujemna lub równa 0: 
let liczba = 1;

if(liczba > 0){
    alert("Liczba jest dodatnia.");
}
else if(liczba < 0){
    alert("Liczba jest ujemna.");
}
else{
    alert("Liczba jest równa 0.");
}

Operatory logiczne - &&, ||

Operatory logiczne pozwalają łączyć wiele warunków w jednej instrukcji warunkowej.

&& - i/oraz, w przypadku, kiedy mamy dwa warunki, oba muszą zostać spełnione.

|| - lub, w przypadku, kiedy mamy dwa warunki, przynjamniej jeden musi zostać spełniony.

Składnia:

&&: 
if(warunek1 && warunek2){
    //kod, który wykona się po spełnieniu dwóch warunków.
}

||: 
if(warunek1 || warunek2){
    //kod, który wykona się po spełnieniu przynajmniej jednego warunku.
}

Przykład:

&&:
Skrypt, który sprawdza, czy wartość zmiennej jest większa od 1 i mniejsza od 10: 
let liczba = 5;

if(liczba>1 && liczba<10){
    alert("Liczba jest poprawna");
}
else{
    alert("Liczba jest spoza zakresu");
}

Pętle - while, do while, for

Pętle pozwalają na wykonanie kodu wielokrotną ilość razy.

while - wykonuje kod jeśli dany warunek jest spełniony.

do while - wykonuje kod przynajmniej raz. Jeśli warunek jest spełniony, wykonuje się dalej.

for - pętla, którą najczęściej wykorzystuje się, gdy chcemy wykonać kod określoną ilość razy.

Składnia:

while: 
while(warunek){
    //Kod wykonywany, dopóki warunek jest prawdziwy.
}
do while: 
do{
    //Kod, który wykonuje się przynajmniej raz.
    //Jeśli warunek jest prawdziwy, wykonuje się dalej.
}while(warunek);
for: 
for(inicjalizacja_zmiennej; warunek; polecenie){
    //Kod do wykonania.
}
  • inicjalizacja_zmiennej - deklaracja oraz ustawienie wartości początkowej zmiennej (najczęściej 0). Zmienna ta będzie licznikiem pętli.
  • warunek - pętla będzie się wykonywać dopóki ten warunek będzie spełniony. Zazwyczaj będzie to po prostu warunek, że nasza zadeklarowana wcześniej zmienna musi być mniejsza od danej przez nas liczby.
  • polecenie - Tutaj podajemy co będzie się działo z naszym licznikiem co każde okrążenie pętli. Najczęściej będzie to inkrementacja, czyli zwiększenie wartości zmiennej o 1.

Przykład:

while:
kod, który do skutku prosi użytkownika o podanie poprawnej odpowiedzi na wynik działania 2+2: 
let odp;
while(odp != 4){
    odp = prompt("Ile to jest 2+2?");
}
for:
kod, który wykona się 5 razy, za każdym razem wypisując do konsoli aktualną wartość i. W konsoli będziemy mogli zobaczyć cyfry od 0-4 (a nie 5), gdyż pętla rozpoczyna od wartości 0 a ostatnią możliwą wartością jest 4 (i ma być mniejsze od 5). Lecz mimo tego kod faktycznie wykonał się 5 razy gdyż:
  • 1 obieg pętli - i = 0
  • 2 obieg pętli - i = 1
  • 3 obieg pętli - i = 2
  • 4 obieg pętli - i = 3
  • 5 obieg pętli - i = 4
for(let i = 0; i < 5; i++){
    console.log(i);
}

Tablice

Tablice są zbiorami danych. Mogą przechowywać dane wszelkiego, dostępnego w JS rodzaju (nawet inne tablice). Deklarujemy je tak samo jak zmienne i stałe.

Składnia:

const tablica1 = [element1, ..., elementX];

Przykład:

let lista_zakupow = ["Mleko", "Chleb", "Jajka", "Sok", "Masło"];

Tablicę stałą możemy modyfikować (omówione dalej) tak samo jak zmienną, lecz nie możemy do niej przypisać innej tablicy tzn.:

const tablica1 = [1,2,3];
let tablica2 = [4,5,6];
//Linijka poniżej nie wykona się i wypisze w konsoli błąd.
tablica1 = tablica2;

Metoda splice:

    splice - 
    
Składnia:

    
tablica.splice(indeks, liczba, element1, ..., elementX);

    
    
        
  • indeks - Indeks elementu od którego chcemy zacząć edytować tablicę.
    • 
        
  • liczba - Liczba elementów, którą chcemy usunąć od elementu z podanym przez nas wcześniej indeksem początkowym (WŁĄCZNIE Z NIM).
    • 
        
  • element1, ..., elementX - Elementy, które chcemy dodać.
    • 
    

    
Przykład:

    Nasza tablica:
    
let lista_zakupow = ["Mleko", "Chleb", "Jajka", "Sok", "Masło"];

    Usuwanie wszystkich elementów z tablicy:
    
lista_zakupow.splice(0);

    Został tutaj zadeklarowany jedynie indeks elementu, od którego chcemy zacząć, w tym przypadku od samego początku tablicy (indeks 0 jest pierwszy). Jeśli nie wprowadziliśmy kolejnego argumentu, czyli liczby elementów, które chcemy usunąć, metoda splice domyślnie usuwa wszystkie następne elementy.


    Usuwanie wszystkiego począwszy od drugiego elementu, który ma indeks 1 (WŁĄCZNIE Z NIM):
    
lista_zakupow.splice(1);
//W tym przypadku w tablicy zostanie jedynie element "Mleko".

    Usuwanie jednego elementu począwszy od elementu trzeciego:
    
lista_zakupow.splice(2, 1);
//W tym przypadku tablica będzie miała postać ["Mleko", "Chleb", "Sok", "Masło"]

    Wstawianie na początku elementów "Piwo", "Gazeta":
    
lista_zakupow.splice(0, 0, "Piwo", "Gazeta");
//Tablica: ["Piwo", "Gazeta", "Mleko", "Chleb", "Jajka", "Sok", "Masło"]

    Nowe elementy zostały wstawione w miejsce elementu pierwszego o indeksie 0, lecz nie został on usunięty ze względu na drugi argument o wartości 0.


    Wstawianie elementów "Piwo", "Gazeta" zaczynając od elementu trzeciego:
    
lista_zakupow.splice(2, 0, "Piwo", "Gazeta");
//Tablica: ["Mleko", "Chleb", "Piwo", "Gazeta", "Jajka", "Sok", "Masło"]

    Usunięcie dwóch elementów i wstawienie "Piwo", "Gazeta" zaczynając od elementu trzeciego:
    
lista_zakupow.splice(2, 2, "Piwo", "Gazeta");
//Tablica: ["Mleko", "Chleb", "Piwo", "Gazeta", "Masło"]

    
Metoda concat - łączenie tablic

    Metoda concat pozwala nam na łączenie tablic oraz dodawanie do nich nowych elementów.
    
Składnia:

    Jeśli chcemy złączyć tablice:
    
const zlaczoneTablice = tablica1.concat(tablica2);

    
    
        
  • zlaczoneTablice - Tablica, do której przypiszemy wartość złączonych tablic - tablica1 i tablica2.
    • 
    

    

    Jeśli chcemy złączyć tablicę i elementy:
    
const tablicaPlusElementy = tablica1.concat(element1, ..., elementX);

    
    
        
  • tablicaPlusElementy - Tablica, do której przypiszemy wartość złączonej tablicy1 z elementami.
    • 
    

    
Przykład:

    Łączenie tablic:
    
const lista_zakupow1 = ["Mleko", "Chleb"];
const lista_zakupow2 = ["Jajka", "Sok"];

const zlaczoneListy = lista_zakupow1.concat(lista_zakupow2);

//Tablica zlaczoneListy będzie miała postać: ["Mleko", "Chleb", "Jajka", "Sok"]
    

    Łączenie tablicy i elementów:
    
const lista_zakupow = ["Mleko", "Chleb"];
       
const lista_zakupow2 = lista_zakupow.concat("Jajka", "Sok");
        
//Tablica lista_zakupow2 będzie miała postać: ["Mleko", "Chleb", "Jajka", "Sok"]
