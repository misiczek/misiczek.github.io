PHP - wprowadzenie

PHP (Hypertext Preprocessor) to skryptowy język programowania zaprojektowany do generowania stron internetowych i budowania aplikacji webowych w czasie rzeczywistym. Działa po stronie serwera, a nie po stronie klienta jak np. JavaScript. Jest językiem interpretowanym, czyli takim, który nie wymaga kompilacji (zamiany kody źródłowego na binarny język maszynowy). Skrypt PHP jest wykonywany na serwerze i zwracany do przeglądarki użytkownika w formie kodu HTML.

Sprawdzenie konfiguracji PHP - phpinfo

phpinfo używamy, jeśli chcemy sprawdzić informacje na temat aktualnej konfiguracji PHP na serwerze.

Przykład:

phpinfo();

Zmienne

Zmienna to miejsce w pamięci komputera, które przechowuje nadaną mu wartość. W PHP nazwy zmiennych zapisuje się z $ na początku.

Składnia:

Przypisanie wartości zmiennej: 
$zmienna = wartosc;

Przykład:

Przypisanie wartości zmiennej: 
$tekst = "Hello world";
$liczba = 1;

Typy danych

Ciąg znaków (string): 
$zmienna = "Hello world";
Liczba całkowita (integer): 
$zmienna = 1;
Liczba zmiennoprzecinkowa (float): 
$zmienna = 5.5;
Wartość logiczna (boolean), prawda: 
$zmienna = true;
Wartość logiczna (boolean), fałsz: 
$zmienna = false;
Oznaczenie braku wartości: 
$zmienna = null;
Tablica (array), (omówione dalej): 
$zmienna = [element1, ..., elementX];

Wypisanie wartości do przeglądarki - echo

Składnia:

echo wartosc;

Przykład:

Wypisanie tekstu: 
echo "Hello world";
Wypisanie zawartości zmiennej: 
echo $zmienna;
Wypisanie znacznika HTML, w tym przypadku nagłówka pierwszego stopnia: 
echo "<h1>Nagłówek</h1>";

Komentarze

Komentarze są krótką informacją do napisanego kodu. Tekst zawarty w komentarzu nie jest interpretowany. W PHP dostępne są komentarze liniowe i blokowe.

Przykład:

//To jest komentarz liniowy. Ogranicza się do jednej linii.

/* To jest
komentarz blokowy.
Może składać się
z wielu linii */

Operatory

Bliźniacze do tych w JavaScripcie, które opisane są tutaj.

Instrukcja warunkowa - if

If sprawdza, czy dany warunek jest prawdziwy, jeżeli tak to wykonuje zawarte polecenie.

elseif - jeśli poprzedni warunek nie został spełniony, elseif sprawdza nowy warunek, jeżeli jest prawdziwy, wykonuje go, w przeciwnym wypadku przechodzi do następnego elseif bądź else. Uwaga: mimo, że w PHP można używać zarówno "elseif" jak i "else if" to zaleca się korzystanie z tego pierwszego!

else - wykonuje kod, jeśli wszystkie poprzednie warunki nie zostały spełnione.

Składnia:

Tylko if: 
if(warunek){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek
}
if, elseif, else: 
if(warunek1){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek1
}
elseif(warunek2){
    //kod, który wykonuje się, jeśli warunek1 nie zostanie spełniony i warunek2 będzie prawdziwy
}
else{
    //kod, który wykonuje się, jeśli wszystkie warunki nie zostały spełnione
}

Przykład:

Skrypt, który sprawdza, czy liczba jest dodatnia, ujemna lub równa 0: 
$liczba = 1;

if($liczba > 0){
    echo "Liczba jest dodatnia.";
}
elseif($liczba < 0){
    echo "Liczba jest ujemna.";
}
else{
    echo "Liczba jest równa 0.";
}

Operatory logiczne - &&/and, ||/or

Operatory logiczne pozwalają łączyć wiele warunków w jednej instrukcji warunkowej.

&& - i/oraz, w przypadku, gdy mamy dwa warunki, oba muszą zostać spełnione.

|| - lub, w przypadku, gdy mamy dwa warunki, przynjamniej jeden musi zostać spełniony.

Uwaga: W PHP możemy zastąpić "&&" słowem "and" i "||" słowem "or".

Składnia:

&&: 
if(warunek1 && warunek2){
    //kod, który wykona się po spełnieniu dwóch warunków.
}

||: 
if(warunek1 || warunek2){
    //kod, który wykona się po spełnieniu przynajmniej jednego warunku.
}

Przykład:

Skrypt, który sprawdza, czy wartość zmiennej jest większa od 1 i mniejsza od 10: 
$liczba = 5;

if($liczba>1 && $liczba<10){
    echo "Liczba jest poprawna";
}
else{
    echo "Liczba jest spoza zakresu";
}

Instrukcja wyboru - switch

Instrukcja switch służy do wykonywania odpowiedniego bloku kodu w zależności od wartości zmiennej.

Składnia:

case - przypadek wartości zmiennej, dla którego wykonuje się dany kod.

break - stosowany, aby PHP nie sprawdzał reszty przypadków po wykonaniu jednego z nich.

default - kod, który wykonuje się, jeśli żaden wcześniejszy przypadek nie zostanie spełniony. Nie musimy po nim pisać break, ponieważ nie ma po nim kolejnych przypadków. 
switch(zmienna){
    case wartosc1:
        //kod do wykonania
        break;
    case ...:
        //kod do wykonania
        break;
    case wartoscX:
        //kod do wykonania
        break;
    default:
        //kod do wykonania
}

Przykład:

Skrypt, który na podstawie zmiennej $dzien wypisuje, czy dzisiaj jest weekend: 
$dzien = "sobota";

switch($dzien){
    case "sobota":
        echo "Dzisiaj jest weekend.";
        break;
    case "niedziela":
        echo "Dzisiaj jest weekend.";
        break;
    default:
        echo "Dzisiaj nie jest weekend.";
}
//Ponieważ wartość zmiennej to "sobota", zostanie wypisany tekst "Dzisiaj jest weekend".

Instrukcja wyboru - match

Składnia:

Do zmiennej $wartoscZwrocona zostanie zwrócona odpowiednia wartoscDoZwrotu na podstawie wartości zmiennej $zmienna. 
$wartoscZwrocona = match($zmienna){
    wartosc1 => wartoscDoZwrotu1,
    ... => ...,
    wartoscX => wartoscDoZwrotuX,
};

Przykład:

Skrypt, który na podstawie wartości zmiennej $owoc zwraca do zmiennej $ktoryOwoc zdanie, który to owoc: 
$owoc = "jablko";

$ktoryOwoc = match($owoc){
    "jablko" => "Ten owoc to jabłko.",
    "banan" => "Ten owoc to banan.",
    "gruszka" => "Ten owoc to gruszka."
};

echo $ktoryOwoc;

//Zostanie wypisane zdanie "Ten owoc to jabłko."

Operator warunkowy - ?

Składnia:

Do zmiennej $odpowiedz zostanie przypisana odpowiednia wartość w zależności od tego, czy warunek jest spełniony. 
$odpowiedz = (warunek) ? wartoscJesliTak : wartoscJesliNie;

Przykład:

$dzien = "poniedzialek";

$czyDzisiajPoniedzialek = ($dzien == "poniedzialek") ? "Tak" : "Nie";

echo $czyDzisiajPoniedzialek;
//W przeglądarce zobaczymy odpowiedź "Tak".

Pętle - while, do while, for, foreach

Pętle umożliwiają wielokrotne wykonanie tego samego fragmentu kodu.

while - Wykonuje kod, jeśli dany warunek jest spełniony.

do while - Wykonuje kod przynajmniej raz. Jeśli warunek jest spełniony, pętla wykonuje się ponownie.

for - Pętla, którą najczęściej wykorzystuje się, gdy chcemy wykonać kod określoną ilość razy.

foreach - Pętla najczęściej stosowana do wykonania kodu dla każdego elementu tablicy.

Składnia:

while: 
while(warunek){
    //Kod wykonywany, dopóki warunek jest prawdziwy.
}
do while: 
do{
    /* Kod, który wykonuje się przynajmniej raz. 
    Jeśli warunek jest spełniony, pętla wykonuje się ponownie. */
}while(warunek);
for:

inicjalizacja_zmiennej - Deklaracja oraz nadanie wartości początkowej zmiennej (najczęściej 0). Zmienna ta pełni rolę licznika pętli.

warunek - Pętla będzie się wykonywać, dopóki ten warunek jest spełniony. Zazwyczaj jest to po prostu warunek, że licznik musi być mniejszy od podanej przez nas liczby, która określa, ile razy pętla ma się wykonać.

polecenie - Tutaj określamy, co będzie się działo z licznikiem przy każdym okrążeniu pętli. Najczęściej jest to inkrementacja, czyli zwiększenie wartości zmiennej o 1.

for(inicjalizacja_zmiennej; warunek; polecenie){
    //Kod do wykonania.
}
foreach:

$element to zmienna przechowującą aktualny element tablicy, do którego możemy się odwołać wewnątrz pętli.

foreach($tablica as $element){
    //Kod do wykonania.
}

Tablice

Tablice są zbiorami danych. Mogą przechowywać dane wszelkiego, dostępnego w PHP typu (nawet inne tablice). Deklarujemy je tak samo jak zmienne. Tablice mogą być indeksowane numerycznie lub asocjacyjne.

Tablica indeksowana numerycznie - Podstawowy typ tablicy w PHP. Każdy element w tablicy posiada indeks w postaci liczby. Pierwszy element w tablicy ma indeks 0, drugi element ma indeks 1 itd.

Tablica asocjacyjna - Ten typ tablicy zamiast indeksów numerycznych używa identyfikatorów znakowych (kluczy).

Składnia:

Deklaracja tablicy (domyślnie indeksowanej numerycznie): 
$tablica = [element1, ..., elementX];
Deklaracja tablicy asocjacyjnej: 
$tablica = ["klucz1" => element1, "..." => ..., "kluczX" => elementX];
Uwaga: przed PHP 5.4 (2012) występowała jedynie składnia poniżej. Obecnie jednak zaleca się używanie tej nowszej. 
$tablica = array(element1, ..., elementX);

Przykład:

Tablica indeksowana numerycznie: 
$lista_zakupow = ["Mleko", "Chleb", "Jajka", "Sok", "Masło"];
Tablica asocjacyjna: 
$osoba = ["imie" => "Jan", "nazwisko" => "Kowalski", "wiek" => 30];

Aby wypisać zawartość tablicy do przeglądarki, możemy użyć print_r (echo w tym przypadku nie zadziała).

Wypisanie całej tablicy:

print_r($tablica);
//W przeciwieństwie do echo, nawiasy są wymagane!

Wypisanie elementu z tablicy o indeksie 1:

print_r($tablica[1]);

Wypisanie elementu z tablicy o kluczu "imie":

print_r($tablica["imie"]);

Metody pracy z tablicami i modyfikacja tablic

Dodawanie jednego lub wielu elementów na początku tablicy: 
array_unshift($tablica, element1, ..., elementX);
Dodawanie jednego lub wielu elementów na końcu tablicy: 
array_push($tablica, element1, ..., elementX);
Usuwa pierwszy element tablicy: 
array_shift($tablica);
Usuwa ostatni element tablicy: 
array_pop($tablica);
Usuwa powtarzające się wartości z tablicy i zwraca nową tablicę: 
$zmienna = array_unique($tablica);
Łączy dwie lub więcej tablic i zwraca nową, połączoną tablicę: 
$zmienna = array_merge($tablica1, ..., $tablicaX);
Sprawdza istnienie podanego klucza w tablicy. Jeśli istnieje, zwraca true, w przeciwnym wypadku zwraca false: 
$zmienna = array_key_exists(nazwa_klucza, $tablica);
Przeszukuje wartości umieszczone w tablicy i zwraca klucz dla pierwszego znalezionego dopasowania. Jeśli zadeklarujemy opcjonalny argument czyDokladnie jako true, dopasowanie musi uwzględniać typ danych (jeżeli nie wpiszemy nic, domyślnie będzie to równoznaczne z false): 
$zmienna = array_search(wartosc, $tablica, czyDokladnie);
Sprawdza, czy dana wartość znajduje się w tablicy. Jeśli tak, zwraca true, w przeciwnym razie zwraca false: 
$zmienna = in_array(wartosc, $tablica);
Zwraca liczbę elementów tablicy: 
$zmienna = count($tablica);
Wybiera losowy element tablicy i zwraca jego klucz. Jeśli podamy opcjonanly argument iloscElementow, zostanie zwrócona tablica z zadeklarowaną przez nas ilością losowych elementow (jeśli nie podamy tego argumenu, zostanie zwrócony tylko jeden element): 
$zmienna = array_rand($tablica, iloscElementow);
Zwraca wartości w tablicy w postaci jednego ciąg znaków (z pominięciem kluczy). Jeśli podamy opcjonalny argument separator, wartości zostaną rozdzielone podanym separatorem np. ", ": 
$zmienna = implode(separator, $tablica);
Zwraca przekształcony ciąg znaków w formie tablicy indeksowanej. Separatorem jest znak oddzielający poszczególne elementy łańcucha np. " " (spacja). Opcjonalny parametr limit ogranicza liczbę elementów tablicy do podanej wartości: 
$zmienna = explode(separator, $tekst, limit);

Sortowanie tablic

Sortuje tablicę rosnąco: 
sort($tablica);
Sortuje tablicę malejąco: 
rsort($tablica);
Sortuje tablicę rosnąco według wartości z zachowaniem kluczy: 
asort($tablica);
Sortuje tablicę malejąco według wartości z zachowaniem kluczy: 
arsort($tablica);
Sortuje tablicę rosnąco według kluczy z zachowaniem wartości: 
ksort($tablica);
Sortuje tablicę malejąco według kluczy z zachowaniem wartości: 
krsort($tablica);

Przykład:

sort: 
$owoce = ["jabłko", "gruszka", "banan"];
sort($owoce);
print_r($owoce);
//Do przeglądarki zostanie zwrócona zawartość tablicy w kolejności: banan, jabłko, gruszka.
rsort: 
$liczby = [12, 53, 89, 34];
rsort($liczby);
print_r($liczby);
//Do przeglądarki zostanie zwrócona zawartość tablicy w kolejności: 89, 53, 34, 12.

Funkcje - function

Funkcje są blokami kodu, który może zostać wykorzystany wielokrotnie w całym skrypcie. Zwarty w funkcji kod jest wykonywany tylko po jej wywołaniu.

Składnia:

Deklaracja funkcji: 
function nazwa_funkcji(parametry){
    //kod, który funkcja ma wykonać.
}
Wywołanie funkcji: 
nazwa_funkcji(parametry);

Przykład:

return - Określa wartość, którą funkcja ma zwrócić. Jeśli wywołanie funkcji przypiszemy do zmiennej, to wtedy wartość zwrócona zostanie wpisana do tej zmiennej. Instrukcja return nie jest obowiązkowa, lecz zazwyczaj chcemy, aby funkcja zwracała jakąś wartość. 
function oblicz_koszt($cena, $liczba){
    return $cena * $liczba;
}

$koszt = oblicz_koszt(3, 5);
//Do zmiennej $koszt zostanie przypisany wynik mnożenia 3 i 5, czyli 15.

Zwracanie wielu wartości

Przykład:

Funkcja przyjmuje dwa parametry, w tym przypadku dwie liczby i zwraca wykonane na nich działania w postaci tablicy asocjacyjnej. 
function kalkulator($a, $b){
    $dzialania = [
    'odejmowanie' => $a - $b,
    'mnozenie' => $a * $b,
    'dzielenie' => $a / $b
    ];

    return $dzialania;
}

$wyniki = kalkulator(5,2);
echo $wyniki['odejmowanie'];
//Zostanie wypisany wynik odejmowania 5 i 2, czyli 3.

Argumenty nazwane (tylko od PHP 8)

Przykład:

Podczas wywoływania funkcji, jej parametry podajemy wraz z ich nazwami (ważne: bez $), przez co nie musimy ich podawać w odpowiedniej kolejności. 
function oblicz_kwote($cena, $liczba){
    return $cena * $liczba;
}

$kwota = oblicz_kwote(liczba: 2, cena: 2);

Deklarowanie typów argumentów i wartości zwracanych

Bardziej zaawansowana rzecz i potrzebna tylko w specyficznych przypadkach, ale warto wspomnieć.

Przykład:

Przy każdym parametrze mamy zadeklarowany jego typ, w tym przypadku liczba całkowita (int - integer). Po nawiasie z parametrami występuje ": int", co oznacza deklarację typu zwracanej przez funkcję wartości. 
function oblicz_kwote(int $cena, int $liczba): int{
    return $cena * $liczba;
}
