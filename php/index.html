PHP - wprowadzenie

PHP (Hypertext Preprocessor) to skryptowy język programowania zaprojektowany do generowania stron internetowych i budowania aplikacji webowych w czasie rzeczywistym. Działa po stronie serwera, a nie po stronie klienta jak np. JavaScript. Jest językiem interpretowanym, czyli takim, który nie wymaga kompilacji (zamiany kody źródłowego na binarny język maszynowy). Skrypt PHP jest wykonywany na serwerze i zwracany do przeglądarki użytkownika w formie kodu HTML.

Sprawdzenie konfiguracji PHP - phpinfo

phpinfo używamy, jeśli chcemy sprawdzić informacje na temat aktualnej konfiguracji PHP na serwerze.

Składnia:

phpinfo();

Zmienne

Składnia:

$zmienna = wartosc;

Przykład:

$tekst = "Hello world";
$liczba = 1;

Typy danych

Ciąg znaków (string): 
$zmienna1 = "Hello world";
Liczba całkowita (intiger): 
$zmienna2 = 1;
Liczba zmiennoprzecinkowa (float): 
$zmienna3 = 5.5;
Wartość logiczna (boolean), prawda: 
$zmienna4 = true;
Wartość logiczna (boolean), fałsz: 
$zmienna5 = false;
Oznaczenie braku wartości: 
$zmienna6 = null;
Tablica (array), (omówione dalej): 
let zmienna7 = array(element1, ..., elementX);

Wypisanie danych do przeglądarki - echo

Składnia:

echo wartosc;

Przykład:

Wypisanie tekstu: 
echo "Hello world";
Wypisanie zawartości zmiennej: 
echo $zmienna;
Wypisanie znacznika HTML, w tym przypadku nagłówka pierwszego stopnia: 
echo "<h1>Nagłówek</h1>";

Operatory

Bliźniacze do tych w JavaScripcie, które opisane są tutaj.

Instrukcja warunkowa - if

if - sprawdza, czy dany warunek jest prawdziwy, jeżeli tak to wykonuje zawarte polecenie.

elseif - jeśli poprzedni warunek nie został spełniony, else if sprawdza nowy warunek, jeżeli jest prawdziwy, wykonuje go, jeżeli jest fałszywy, przechodzi do następnego elseif bądź else. Uwaga: mimo, że w PHP można używać zarówno "elseif" jak i "else if" to zaleca się korzystanie z tego pierwszego!

else - wykonuje kod, jeśli wszystkie poprzednie warunki nie zostały spełnione.

Składnia:

Tylko if: 
if(warunek){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek
}
if, elseif, else: 
if(warunek1){
    //kod, który wykonuje się, jeśli zostanie spełniony warunek1
}
elseif(warunek2){
    //kod, który wykonuje się, jeśli warunek1 nie zostanie spełniony i warunek2 zostanie spełniony
}
else{
    //kod, który wykonuje się, jeśli wszystkie warunki nie zostaną spełnione
}

Przykład:

Kod, który sprawdza, czy liczba jest dodatnia, ujemna lub równa 0: 
$liczba = 1;

if($liczba > 0){
    echo("Liczba jest dodatnia.");
}
elseif($liczba < 0){
    echo("Liczba jest ujemna.");
}
else{
    echo("Liczba jest równa 0.");
}

Operatory logiczne - &&/and, ||/or

Operatory logiczne pozwalają łączyć wiele warunków w jednej instrukcji warunkowej.

&& - i/oraz, w przypadku, kiedy mamy dwa warunki, oba muszą zostać spełnione.

|| - lub, w przypadku, kiedy mamy dwa warunki, przynjamniej jeden musi zostać spełniony.

Uwaga: W PHP możemy także zastąpić "&&" słowem "and" i "||" słowem "or".

Składnia:

&&: 
if(warunek1 && warunek2){
    //kod, który wykona się po spełnieniu dwóch warunków.
}

||: 
if(warunek1 || warunek2){
    //kod, który wykona się po spełnieniu przynajmniej jednego warunku.
}

Przykład:

Skrypt, który sprawdza, czy wartość zmiennej jest większa od 1 i mniejsza od 10: 
$liczba = 5;

if($liczba>1 && $liczba<10){
    echo "Liczba jest poprawna";
}
else{
    echo "Liczba jest spoza zakresu";
}

Instrukcja wyboru - switch

Switch używany jest do wykonywania odpowiedniego bloku kodu na podstawie wartości zmiennej.

Składnia:

switch(zmienna){
    case wartosc1:
        //kod do wykonania
        break;
    case ...:
        //kod do wykonania
        break;
    case wartoscX:
        //kod do wykonania
        break;
    default:
        //kod do wykonania
}
  • default - kod, który wykonuje się, jeśli żaden wcześniejszy przypadek nie zostanie spełniony.

Przykład:

Skrypt, który na podstawie zmiennej $dzien wypisuje, czy dzisiaj jest weekend: 
$dzien = "sobota";

switch($dzien){
    case "sobota":
        echo "Dzisiaj jest weekend.";
        break;
    case "niedziela":
        echo "Dzisiaj jest weekend.";
        break;
    default:
        echo "Dzisiaj nie jest weekend.";
}

Instrukcja wyboru - match

Składnia:

$wartoscZwrocona = match($zmienna){
    wartoscZmiennej1 => wartoscDoZwrotu,
    ... => wartoscDoZwrotu,
    wartoscZmiennejX => wartoscDoZwrotu,
};
Do zmiennej $wartoscZwrocona zostaje zwrócona odpowiednia wartoscDoZwrotu na podstawie wartości zmiennej $zmienna.

Przykład:

Kod, który na podstawie zmiennej $owoc zwraca nam do zmiennej $ktoryOwoc zdanie, który to owoc: 
$owoc = "jablko";

$ktoryOwoc = match($owoc){
    "jablko" => "Ten owoc to jabłko.",
    "banan" => "Ten owoc to banan.",
    "gruszka" => "Ten owoc to gruszka."
};

echo $ktoryOwoc;

//W przeglądarce zobaczymy "Ten owoc to jabłko."

Operator warunkowy - ?

Składnia:

$odpowiedz = (warunek) ? odpowiedzJesliTak : odpowiedzJesliNie;

Przykład:

$dzien = "poniedzialek";

$czyDzisiajPoniedzialek = ($dzien == "poniedzialek") ? "Tak" : "Nie";

echo $czyDzisiajPoniedzialek;
//W przeglądarce zobaczymy odpowiedź "Tak".

Pętle - while, do while, for, foreach

Pętle pozwalają na wykonanie kodu wielokrotną ilość razy.

while - wykonuje kod jeśli dany warunek jest spełniony.

do while - wykonuje kod przynajmniej raz. Jeśli warunek jest spełniony, wykonuje się dalej.

for - pętla, którą najczęściej wykorzystuje się, gdy chcemy wykonać kod określoną ilość razy.

foreach - pętla najczęściej stosowana do wykonania danego kodu dla każdego elementu tablicy.

Składnia:

while: 
while(warunek){
    //Kod wykonywany, dopóki warunek jest prawdziwy.
}
do while: 
do{
    //Kod, który wykonuje się przynajmniej raz.
    //Jeśli warunek jest prawdziwy, wykonuje się dalej.
}while(warunek);
for: 
for(inicjalizacja_zmiennej; warunek; polecenie){
    //Kod do wykonania.
}
  • inicjalizacja_zmiennej - deklaracja oraz ustawienie wartości początkowej zmiennej (najczęściej 0). Zmienna ta będzie licznikiem pętli.
  • warunek - pętla będzie się wykonywać dopóki ten warunek będzie spełniony. Zazwyczaj będzie to po prostu warunek, że nasza zadeklarowana wcześniej zmienna musi być mniejsza od danej przez nas liczby określającej ile razy pętla ma się wykonać.
  • polecenie - Tutaj podajemy co będzie się działo z naszym licznikiem co każde okrążenie pętli. Najczęściej będzie to inkrementacja, czyli zwiększenie wartości zmiennej o 1.
foreach: 
foreach($tablica as $element){
    //Kod do wykonania.
}

Tablice - array

Tablice są zbiorami danych. Mogą przechowywać dane wszelkiego, dostępnego w PHP rodzaju (nawet inne tablice). Deklarujemy je tak samo jak zmienne.

Składnia:

$zmienna = array(element1, ..., elementX);

Przykład:

$lista_zakupow = array("Mleko", "Chleb", "Jajka", "Sok", "Masło");
Aby zwrócić zawartość tablicy do przeglądarki, możemy użyć print_r (echo w tym przypadku nie zadziała): 
print_r($tablica);
//W przeciwieństwie do echo, nawiasy są wymagane!

Sortowanie tablic - sort, rsort

sort - sortuje tablicę rosnąco.

rsort - sortuje tablicę malejąco.

Składnia:

sort: 
sort($tablica);
rsort: 
rsort($tablica);

Przykład:

sort: 
$owoce = array("jabłko", "gruszka", "banan");
sort($owoce);
print_r($owoce);
//Do przeglądarki zostanie zwrócona zawartość tablicy w kolejności: banan, jabłko, gruszka.
rsort: 
$liczby = array(12, 53, 89, 34);
rsort($liczby);
print_r($liczby);
//Do przeglądarki zostanie zwrócona zawartość tablicy w kolejności: 89, 53, 34, 12.
