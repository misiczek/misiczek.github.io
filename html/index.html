HTML - wprowadzenie

HTML (HyperText Markup Language) to język znaczników stosowany do tworzenia i prezentowania stron internetowych w przeglądarkach. Opisuje strukturę informacji witryn i początkowo został zaprojektowany jako język opisu dokumentów tekstowych z możliwością dodawania odnośników. HTML umożliwia strukturyzację treści na stronie: definiowanie nagłówków, paragrafów, list, odnośników oraz innych elementów.

Przykład:

<p>Tekst w paragrafie</p>

Linia pozioma - <hr>

Przykład:

<hr>

    Listy

    <ul> - Lista nieuporządkowana (punktowana)

    <ol> - Lista numerowana

    <li> - Element listy

    <dl> - Lista definicji

    <dt> - Termin/nazwa na liście definicji

    <dd> - Opis terminu/nazwy

    Przykład:

    Lista nieuporządkowana (punktowana): 
    <ul>
    <li>to</li>
    <li>jest</li>
    <li>lista</li>
    <li>ul</li>
</ul>
    • to
    • jest
    • lista
    • ul

    Lista numerowana: 
    <ol>
    <li>to</li>
    <li>jest</li>
    <li>lista</li>
    <li>ol</li>
</ol>
    1. to
    2. jest
    3. lista
    4. ol

    Lista definicji: 
    <dl>
    <dt>to</dt>
    <dd>jest</dd>
    <dt>lista</dt>
    <dt>dl</dt>
</dl>
    to
    jest
    lista
    dl
